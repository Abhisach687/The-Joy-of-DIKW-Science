# The Joy of DIKW Science

This project teaches the Data-Information-Knowledge-Wisdom (DIKW) hierarchy and other data and information science concepts through interactive visualizations and code blocks.

## How the App Works

This is an adaptive learning platform that guides you through data science concepts from basic data handling to advanced machine learning and ethics. The app runs entirely in your browser and features:

### Lesson Structure
Each lesson follows a consistent format:
- **Short Explainer**: A concise introduction to the concept.
- **Visual/Diagram**: Interactive charts and visualizations using Plotly.js.
- **Code Example**: Python code snippets that demonstrate the concept.
- **Exercise**: Hands-on coding challenges where you write and test your own code.

### Adaptive Learning
The app adjusts to your learning pace:
- **Difficulty Modes**: Guided, Assisted, Independent, and Challenge levels.
- **Hint System**: Progressive hints that reveal as needed.
- **XP and Progress Tracking**: Earn experience points and track completion.
- **Flashcards**: Spaced repetition system for reinforcing key concepts.

### Key Features
- **Browser Python**: Run Python code directly in the browser using Pyodide (no installation required).
- **Local Storage**: All progress is saved locally in your browser.
- **Project Ladder**: Build practical projects to apply your knowledge.
- **DIKW Lens**: Each lesson connects to the Data-Information-Knowledge-Wisdom framework.

### Getting Started
1. Open `index.html` in a modern web browser.
2. Start with the first lesson in the sidebar.
3. Run code examples and complete exercises.
4. Use flashcards for review and projects for deeper learning.
